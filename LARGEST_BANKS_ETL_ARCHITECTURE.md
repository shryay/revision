## 1. What this project is

This project is a small **data engineering ETL pipeline** that:

- **Extracts**: scrapes the “List of largest banks” table from Wikipedia (via an archived snapshot URL)
- **Transforms**: converts each bank’s **Market Cap (USD, billions)** into **GBP/EUR/INR (billions)** using `exchange_rate.csv`
- **Loads**:
  - writes a final dataset to `Largest_banks_data.csv`
  - writes the same dataset into a SQLite database `Banks.db` as table `Largest_banks`
- **Validates**: runs a few SQL queries (full table, average market cap in GBP, top 5 names)
- **Logs**: writes timestamped stage messages to `code_log.txt`

**Important**: There is **no frontend/backend web app** here. It’s a **batch ETL script** (`banks_project.py`) plus an equivalent notebook (`WorldBank_Project.ipynb`).

Low level design (LLD)\n+\n+The implementation is a single script with small focused functions.\n+\n+### 4.1 Function-level design\n+\n+```mermaid\n+classDiagram\n+  class banks_project {\n+    +log_progress(message)\n+    +extract(url, table_attribs) DataFrame\n+    +transform(df, csv_path) DataFrame\n+    +load_to_csv(df, output_path)\n+    +load_to_db(df, sql_connection, table_name)\n+    +run_query(query_statement, sql_connection)\n+  }\n+```\n+\n+### 4.2 `log_progress(message)`\n+- **Responsibility**: append timestamped stage messages to `code_log.txt`\n+- **Format**: `<YYYY-Mon-DD-HH:MM:SS> : <message>`\n+- **Why it exists**: makes the ETL auditable and easy to debug.\n+\n+### 4.3 `extract(url, table_attribs)`\n+- **Input**: archived Wikipedia URL, list of columns\n+- **Steps**:\n+  - `requests.get(url).text`\n+  - `BeautifulSoup(html, \"html.parser\")`\n+  - find the first `<tbody>` and iterate `<tr>` rows\n+  - for each row:\n+    - name from `col[1].find_all(\"a\")[1][\"title\"]`\n+    - market cap from `col[2].contents[0]` with trimming (e.g., `432.92\\n`)\n+- **Output**: DataFrame with columns:\n+  - `Name`\n+  - `MC_USD_Billion`\n+\n+### 4.4 `transform(df, csv_path)`\n+- **Input**: extracted DataFrame + `exchange_rate.csv`\n+- **Steps**:\n+  - read CSV → dict `{\"EUR\": 0.93, \"GBP\": 0.8, \"INR\": 82.95}`\n+  - compute:\n+    - `MC_GBP_Billion = round(MC_USD_Billion * GBP, 2)`\n+    - `MC_EUR_Billion = round(MC_USD_Billion * EUR, 2)`\n+    - `MC_INR_Billion = round(MC_USD_Billion * INR, 2)`\n+- **Output**: enriched DataFrame (5 columns total)\n+\n+### 4.5 `load_to_csv(df, output_path)`\n+- Writes DataFrame to `Largest_banks_data.csv`\n+\n+### 4.6 `load_to_db(df, sql_connection, table_name)`\n+- Writes DataFrame into SQLite using `df.to_sql(..., if_exists='replace', index=False)`\n+- Table name: `Largest_banks`\n+\n+### 4.7 `run_query(query_statement, sql_connection)`\n+- Uses `pd.read_sql(query_statement, sql_connection)` and prints results\n+- Queries executed by the script:\n+  - `SELECT * from Largest_banks`\n+  - `SELECT AVG(MC_GBP_Billion) FROM Largest_banks`\n+  - `SELECT Name from Largest_banks LIMIT 5`\n+\n+---\n+\n+## 5.
